# C Programming Exercises

This folder contains practical exercises developed to build a strong foundation in procedural programming, algorithmic logic, and low-level memory management using the C language. 

The exercises are divided into specific academic units, showing a clear progression from basic data manipulation to advanced memory and file management.

---

## 🚀 Key Competencies Demonstrated

* **Memory Management & Pointers:** Direct memory manipulation, passing by reference, and dynamic memory allocation using `malloc` and `free`.
* **Data Structures:** Defining and utilizing custom `structs` to organize data and build more complex registry systems.
* **File Handling (I/O):** Reading from and writing to both text (`.txt`) and binary (`.dat`) files for data persistence and searching.
* **Core Logic & Algorithms:** Control structures, modular programming with functions, and advanced array/string manipulation (e.g., custom string length counting and palindrome verification).
