# C Programming Exercises

This folder contains practical exercises developed to build a strong foundation in procedural programming, algorithmic logic, and low-level memory management using the C language.

---

## 🚀 Key Competencies Demonstrated

* **Memory Management & Pointers:** Direct memory manipulation, passing by reference, and dynamic memory allocation using `malloc` and `free`.
* **Data Structures:** Defining and utilizing custom `structs` to organize data and build more complex systems.
* **File Handling (I/O):** Reading from and writing to both text (`.txt`) and binary (`.bin` / `.dat`) files for data persistence.
* **Core Logic & Algorithms:** Control structures, loops, modular programming with functions, and array/string manipulation.
