# SQL & Database Management Exercises

This folder contains practical scripts, relational database models, and queries developed during my course. The focus here is on designing structured relational databases, ensuring data integrity, and writing optimized queries using MySQL.
It also contains de final exam.

---

## 🚀 Key Competencies Demonstrated

* **Data Definition Language (DDL):** Creating, altering, and dropping tables with appropriate data types, constraints (`PRIMARY KEY`, `FOREIGN KEY`, `UNIQUE`), and managing `AUTO_INCREMENT` behaviors.
* **Data Manipulation Language (DML):** Inserting, updating, and deleting records safely while maintaining referential integrity.
* **Advanced Querying:** Writing efficient data retrieval scripts using `INNER/LEFT/RIGHT JOINs`, subqueries, aggregate functions (`COUNT`, `SUM`, `AVG`), and data grouping (`GROUP BY`, `HAVING`).
* **Database Modeling:** Translating logical requirements into physical database schemas, implementing proper relationships (1:1, 1:N, N:M), and applying normalization principles.
