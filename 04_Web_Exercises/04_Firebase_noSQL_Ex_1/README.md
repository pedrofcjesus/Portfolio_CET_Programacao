# Gestor de Produtos

Este é um exercício simples de gestão de produtos (CRUD) desenvolvido com **HTML, CSS e JavaScript**, feito com o objetivo de praticar a ligação à base de dados NoSQL do **Firebase**.

### 🔒 Nota sobre a API Key
Por motivos de segurança, a API key do Firebase foi removida deste repositório público. 

No entanto, para que possa ver a aplicação a funcionar imediatamente, adicionei uma função que guarda e lê os dados diretamente no **LocalStorage** do browser sempre que a chave do Firebase não estiver configurada.
https://pedrofcjesus.github.io/Portfolio_CET_Programacao/04_Web_Exercises/04_Firebase_noSQL_Ex_1/index.html


### 🛠️ Como testar localmente
1. Clone o repositório.
2. Abra o projeto através de um servidor local (como a extensão **Live Server** do VS Code), uma vez que o código utiliza módulos de JavaScript.
