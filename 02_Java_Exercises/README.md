# Java Programming Exercises

This folder contains practical exercises developed to build a solid foundation in Object-Oriented Programming (OOP) using Java. The projects range from basic syntax and logic to class inheritance and object composition.

---

## 🚀 Key Competencies Demonstrated

* **Core Syntax & Control Flow:** Standard I/O operations (`Scanner`), string formatting/escape characters, conditionals (`if/else`), and iterative loops (`for`, `while`).
* **Object-Oriented Principles:** Creation of classes, instantiation of objects, and defining methods to encapsulate business logic (e.g., bank account operations, invoicing).
* **Object Composition:** Establishing relationships between different classes (e.g., linking a `CLIENTE` to a `CARTAODECREDITO`, or an `AGENCIA` to a `CONTA`).
* **Inheritance:** Utilizing the `extends` keyword to inherit attributes and methods from a parent class (e.g., a subclass inheriting from `PESSOA`), promoting code reusability.
